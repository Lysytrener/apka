# Łysy Trener App — wersja z atlasem z Excela

## Co jest nowe
- formularz dodawania podopiecznych do Supabase,
- atlas ćwiczeń ładowany z tabeli `exercises`,
- import SQL: `supabase/import_cwiczenia_lysy_trener.sql`.

## Jak załadować ćwiczenia
1. Supabase → SQL Editor → New Query.
2. Wklej zawartość pliku `supabase/import_cwiczenia_lysy_trener.sql`.
3. Kliknij Run.
4. Zrób redeploy w Vercel.
